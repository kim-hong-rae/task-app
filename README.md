# 과제

## 기술 스택

- **React** - 사용자 인터페이스를 구축하기 위한 JavaScript 라이브러리
- **TypeScript** - JavaScript의 타입 확장 언어
- **Styled-Components** - 스타일을 컴포넌트 단위로 관리할 수 있는 CSS-in-JS 라이브러리
- **Vite** - 빠르고 간편한 웹 개발을 위한 빌드 도구

## 프로젝트 실행 방법

1. **프로젝트 클론하기**
2. **npm install**
3. **npm run dev**
